
# Node API

Learnt how to build a RESTful CRUD API with Node.js,Express and MongoDB from scratch.

NPM packages used : Express,Mongoose,CORS,dotenv,asyncHandler



## API Reference

#### Get all items

  GET /api/products

#### Get item

  GET /api/products/{id}

#### POST an item

  POST /api/products

#### update an item

  PUT /api/products/{id}

#### Delete an item

  Delete /api/products/{id}
